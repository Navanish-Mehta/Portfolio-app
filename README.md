# Modern Portfolio Website

A modern, responsive portfolio website built with React, TypeScript, and Styled Components. This portfolio showcases your work, skills, and experience in a beautiful and interactive way.

## Features

- 🎨 Modern and clean design
- 📱 Fully responsive layout
- ⚡ Smooth animations and transitions
- 🌙 Dark theme optimized
- 🎯 SEO friendly
- 🚀 Fast performance
- 📝 Blog section
- 📬 Contact form
- 🔗 Social media integration

## Tech Stack

- React 18
- TypeScript
- Styled Components
- Framer Motion
- React Icons
- Three.js (for 3D elements)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio-app.git
```

2. Install dependencies:
```bash
cd portfolio-app
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project Structure

```
src/
  ├── components/     # Reusable components
  ├── sections/       # Page sections
  ├── styles/         # Global styles
  ├── assets/         # Images and other assets
  ├── App.tsx         # Main App component
  └── index.tsx       # Entry point
```

## Customization

1. Update personal information in the respective section components
2. Replace images in the `public/images` directory
3. Modify colors and styles in the styled components
4. Add or remove sections as needed

## Deployment

1. Build the project:
```bash
npm run build
```

2. Deploy the `build` folder to your preferred hosting service (e.g., Netlify, Vercel, GitHub Pages)
